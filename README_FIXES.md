# Правки v14 — исправление ошибки на вкладке «О нас»

## Что исправлено

1. `/about` больше не падает, если в `/api/home` нет `about_title` или `about_content`.
2. Страница «О нас» теперь безопасно использует резервный текст и проверяет, что сотрудники пришли массивом.
3. `/api/home` теперь возвращает дефолтные поля `about_title` и `about_content`.
4. `/api/home` теперь корректно принимает сохранение в формате массива `{ key, value }`, который использует админка «О нас».

## Как применить

Распаковать архив в корень проекта с заменой файлов, затем:

```cmd
cd C:\Users\User\Desktop\kino75-main
pnpm build
git add -A
git commit -m "Fix about page on Vercel"
git push
```

После push Vercel сам начнет новый деплой. Если не начнет — в Vercel открой Deployments и нажми Redeploy.
