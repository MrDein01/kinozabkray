# Правки v15 — центрирование редактора новостей

Исправлено отображение страницы `/admin/news`:

1. Список новостей остаётся широким.
2. Форма добавления/редактирования новости больше не прижимается к левому краю.
3. Вкладки «Редактирование / Предпросмотр», форма и предпросмотр теперь центрируются по странице.

## Как применить

Распакуйте архив в корень проекта с заменой файлов, затем выполните:

```cmd
cd C:\Users\User\Desktop\kino75-main
pnpm build
git add -A
git commit -m "Center admin news editor"
git push
```

После `git push` Vercel автоматически начнёт новый деплой.
