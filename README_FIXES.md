# Правки v17 — загрузка фото сотрудников на Vercel

## Что исправлено

1. Исправлена загрузка фото сотрудника в админке.
2. API `/api/upload` теперь возвращает сразу два поля: `url` и `fileUrl`.
3. Новости продолжат брать `url`, а сотрудники теперь корректно получают `fileUrl`.
4. Файлы не сохраняются в `public/uploads`, а кодируются в data URL и записываются в базу Neon вместе с записью сотрудника/новости.
5. Максимальный размер изображения — 4 МБ.

## Важно про новости

Этот фикс НЕ трогает базу и НЕ перезаполняет новости. Чтобы сохранить новости, которые уже были сделаны на сайте, НЕ запускайте:

```cmd
pnpm exec ts-node --compiler-options "{\"module\":\"CommonJS\"}" scripts/apply-manual-content.ts
```

После этого архива нужно только собрать, закоммитить и отправить код в GitHub.

## Как применить

Распакуйте архив в корень проекта с заменой файлов:

```cmd
cd C:\Users\User\Desktop\kino75-main
pnpm build
git add -A
git commit -m "Fix employee photo upload on Vercel"
git push
```

Vercel после `git push` должен сам запустить новый деплой.
