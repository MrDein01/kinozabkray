# Правки v13 — настоящая страница списка новостей в админке

## Что сделано

1. `/admin/news` теперь при первом открытии показывает список всех новостей, как на скрине:
   - верхняя панель: «Назад», «Управление новостями», «+ Добавить новость»;
   - заголовок «Все новости»;
   - справа счётчик «Всего: ...»;
   - новости карточками с картинкой, статусом, датой, заголовком, описанием, ссылкой;
   - справа иконки редактирования и удаления.
2. Форма создания новости больше не открывается сразу при входе во вкладку.
3. Кнопка «+ Добавить новость» открывает форму создания.
4. Иконка карандаша открывает редактирование выбранной новости.
5. Кнопка «Назад» на списке возвращает в `/admin`, а в редакторе возвращает к списку новостей.
6. Загрузка картинки с компьютера и предпросмотр новости сохранены.

## Команды после распаковки

cd C:\Users\User\Desktop\kino75-main
