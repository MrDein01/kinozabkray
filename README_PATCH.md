# Правки от 2026-05-23

## Что изменено

1. **Подвал / соцсети**
   - `components/Footer.tsx` теперь выводит массив соцсетей, а не одну жёстко заданную VK-кнопку.
   - Поддерживается хранение `footer_socials` как JSON-массива, объекта или простого списка строк.
   - Соцсети показываются рядом и автоматически переносятся на следующую строку, если места мало.
   - Добавлены дефолтные ссылки: ВКонтакте и Telegram.

2. **API главной страницы**
   - `app/api/home/route.ts` сохраняет `footer_socials` массивом через JSON, чтобы вторая соцсеть не затирала первую на уровне API/данных.

3. **Разделы сайта**
   - `app/anti-corruption/page.tsx` теперь показывает справочную информацию, нормативные документы и контакты.
   - `app/prosecutor/page.tsx` теперь показывает правовые разъяснения и дополняет существующие записи базовыми материалами.

4. **Статистика**
   - `components/VisitTracker.tsx` отправляет визиты через `sendBeacon`/`fetch keepalive` и не ломает интерфейс при ошибках.
   - `app/api/statistics/route.ts` безопасно пишет/читает статистику и возвращает `200`, даже если таблица временно недоступна.
   - `app/admin/statistics/page.tsx` отрисовывает статистику без тяжёлых клиентских графиков и показывает понятное сообщение при проблеме с БД.
   - `middleware.ts` больше не пытается писать статистику из Edge Runtime.

5. **База данных**
   - `prisma/init.sql` дополнен таблицей `visit_stats`, индексами, контактами, соцсетями и начальными материалами.
   - Для существующей базы добавлен быстрый SQL-фикс: `deploy/sql/2026-05-23-fix-visit-stats.sql`.
   - `prisma/seed.ts` дополнен контактами подвала, соцсетями и материалами раздела «Прокурор разъясняет».

## Как применить

Скопируйте файлы из архива в корень проекта с заменой.

Для существующей базы выполните один из вариантов:

```bash
pnpm db:push
```

или только SQL-фикс:

```bash
psql "$DATABASE_URL" -f deploy/sql/2026-05-23-fix-visit-stats.sql
```

Потом пересоберите и перезапустите приложение:

```bash
pnpm build
pm2 restart kino75
```

Если запускаете локально:

```bash
pnpm dev
```
